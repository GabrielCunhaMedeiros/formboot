# 📋 Declaração de Residência

Este projeto consiste em um formulário HTML para preenchimento de uma **Declaração de Residência**, utilizando a biblioteca Bootstrap 5 para estilização e organização dos elementos.

## 🔧 Tecnologias Utilizadas

* HTML5
* Bootstrap 5 (via CDN)

## 📷 Interface

O formulário inclui os campos necessários para a identificação do declarante e a confirmação de seu endereço residencial.

Todos os campos estão organizados utilizando a grid system do Bootstrap (`.row`, `.col-md-*`), proporcionando uma estrutura limpa e organizada.

## 📦 Objetivo

O objetivo deste formulário é permitir o preenchimento digital de uma Declaração de Residência utilizada em processos do DETRAN, possibilitando que o cidadão informe e formalize seu endereço residencial quando necessário para comprovação de residência em procedimentos administrativos.
